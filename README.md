# Bulb Toggle Project

A simple and fun JavaScript mini project that demonstrates DOM manipulation using addEventListener. Click the button to turn the bulb *ON* or *OFF* â€” when it's ON, the background turns black and the glowing bulb creates a realistic light effect. 

This project is also *fully responsive* and works well on all screen sizes.

## Features

- Toggle bulb ON/OFF with a button click
- Background turns black when the bulb is ON
- Glowing effect using CSS box-shadow
- Smooth transitions and light scaling
- Responsive design
